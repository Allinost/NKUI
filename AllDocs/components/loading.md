# Loading 加载

加载状态指示器。

## 尺寸

<ClientOnly>
<Demo>
  <Space>
    <Loading size="sm" />
    <Loading size="md" />
    <Loading size="lg" />
  </Space>
</Demo>
</ClientOnly>

## 条形加载

<ClientOnly>
<Demo>
  <Loading variant="bar" />
</Demo>
</ClientOnly>

## API

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | 加载图标尺寸 |
| variant | `'spinner' \| 'bar'` | `'spinner'` | 加载样式变体 |
