# 间距与布局 (Spacing & Layout)

## 间距系统

以 4px 为基准单位的递增系统。

| Token | Value | Example |
|-------|-------|---------|
| `nkui.spacing.xs` | 4px | 图标与文字间距 |
| `nkui.spacing.sm` | 8px | 内边距(小) |
| `nkui.spacing.md` | 12px | 内边距(中)、组件间距 |
| `nkui.spacing.lg` | 16px | 卡片内边距 |
| `nkui.spacing.xl` | 24px | 区块间距 |
| `nkui.spacing.2xl` | 32px | 大区块间距 |
| `nkui.spacing.3xl` | 48px | 页面边距 |

## 圆角

| Token | Value | Usage |
|-------|-------|-------|
| `nkui.radius.none` | 0 | |
| `nkui.radius.sm` | 4px | 输入框 |
| `nkui.radius.md` | 8px | 卡片、按钮 |
| `nkui.radius.lg` | 12px | 对话框 |
| `nkui.radius.xl` | 16px | 大卡片 |
| `nkui.radius.full` | 9999px | 胶囊按钮、Tag |

## 断点 (Breakpoints)

| Token | Value | Device |
|-------|-------|--------|
| `nkui.breakpoint.sm` | 640px | Mobile |
| `nkui.breakpoint.md` | 768px | Tablet |
| `nkui.breakpoint.lg` | 1024px | Desktop |
| `nkui.breakpoint.xl` | 1280px | Wide |
| `nkui.breakpoint.2xl` | 1536px | Ultra-wide |
