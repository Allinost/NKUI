# 字体排版 (Typography)

## 字体栈

| Token | Value |
|-------|-------|
| `nkui.font.family.sans` | `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 'Noto Sans', sans-serif` |
| `nkui.font.family.mono` | `'SF Mono', 'Fira Code', 'Fira Mono', 'Roboto Mono', 'Courier New', monospace` |

## 字号层级

| Token | Size | Usage |
|-------|------|-------|
| `nkui.font.size.xs` | 10px | 辅助/角标文字 |
| `nkui.font.size.sm` | 12px | 次要说明 |
| `nkui.font.size.base` | 14px | **正文默认** |
| `nkui.font.size.lg` | 16px | 加强正文 |
| `nkui.font.size.xl` | 20px | 小标题 |
| `nkui.font.size.2xl` | 24px | 标题 |
| `nkui.font.size.3xl` | 32px | 大标题 |

## 字重

| Token | Value | Usage |
|-------|-------|-------|
| `nkui.font.weight.regular` | 400 | 正文 |
| `nkui.font.weight.medium` | 500 | 强调 |
| `nkui.font.weight.semibold` | 600 | 小标题 |
| `nkui.font.weight.bold` | 700 | 标题 |

## 行高

| Token | Value | Usage |
|-------|-------|-------|
| `nkui.font.lineHeight.tight` | 1.2 | 标题 |
| `nkui.font.lineHeight.normal` | 1.5 | 正文 |
| `nkui.font.lineHeight.relaxed` | 1.75 | 长文阅读 |
